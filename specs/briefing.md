The task is to make the absence request processor production ready. Employees request time off, and the script needs to check entitlement, approve or reject requests, update balances, and post the decision to the HR API.

A few helpful hints:

- Start by reading docs/LEAVE_POLICY.md carefully — that defines what “correct” means.
- Before coding, please create QUESTIONS.md with your questions and assumptions.
- Then write a short SPEC.md explaining the rules, edge cases, what’s out of scope, and how you’ll test it.
- Tests matter here, so extend the test suite as you go.
